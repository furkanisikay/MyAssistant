# Bağımlılık Dosyaları

Bu klasöre `Optimizasyon.dll` dosyasını yerleştirmeniz gerekmektedir.

`Optimizasyon.dll`, [NHTools](https://github.com/furkanisikay) kütüphanesinin bir parçasıdır ve projenin derlenmesi için gereklidir.

## Kurulum

1. `Optimizasyon.dll` dosyasını bu klasöre (`lib/`) kopyalayın.
2. Projeyi Visual Studio ile açın ve derleyin.
